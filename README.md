# SWJTU 网络学堂自动化工具集

针对西南交通大学网络学堂（e-learning.swjtu.edu.cn）开发的一组自动化辅助工具，基于 CDP + Playwright + customtkinter 实现。

---

## 项目列表

| 目录 | 功能 |
|------|------|
| `swjtu-course-python` | 课程学习助手（主力版） |
| `swjtu-course-panel` | 课程学习助手（旧版） |
| `swjtu-quiz-filler` | 客观题自动答题助手 |

---

## swjtu-course-python — 课程学习助手

自动打开课程文档/视频条目，完成学习进度记录。

### 功能

- 连接已打开的 Chrome（CDP 协议），或一键自动启动 Chrome
- 读取当前课程页面的所有条目（文档 / 视频）
- 支持按学习状态（已学习 / 未学习 / 全部）和类型（doc / video）过滤
- 逐条自动点击，支持全选 / 全不选 / 反选
- 可配置打开间隔、关闭间隔、单项超时
- 支持暂停 / 继续 / 停止

### 使用步骤

```bash
cd swjtu-course-python
pip install -r requirements.txt
playwright install chromium
python app.py
```

1. 在浏览器中打开课程目录页
2. 点击「一键启动并连接」或「连接浏览器」
3. 设置过滤条件，点击「刷新条目」
4. 勾选要执行的条目，点击「开始」

## swjtu-course-panel — 课程学习助手（旧版）

功能与 `swjtu-course-python` 相同，区别在于条目列表为纯文本显示，不支持单独勾选，只能全量执行。保留作为参考对比。

## swjtu-quiz-filler — 客观题自动答题助手

读取答题页面的单选题和多选题，根据导入的答案文件自动点击选项。

### 功能

- 扫描当前答题页面，识别所有单选题 / 多选题（基于 Element UI `div.list-item` 结构）
- 按行读取答案文件，逐题自动点击对应选项
- 完成后可验证选中状态（显示 ✓ / ✗）
- 不自动提交，人工确认后手动点提交

### 答案文件格式

每行对应一道题，第 N 行 = 第 N 题：

```
存储程序与自动控制
通用计算机和专用计算机
由于计算机的可编程性，计算机能够在各行各业得到广泛的应用
内存储器；I/O设备；中央处理器
```

- **单选题**：直接写选项文字
- **多选题**：用 `；`（中文分号）分隔多个答案
- **填空题 / 主观题**：留空，程序自动跳过

### 使用步骤

```bash
cd swjtu-quiz-filler
pip install -r requirements.txt
playwright install chromium
python app.py
```

1. 在浏览器中打开答题页面
2. 点击「一键启动并连接」或「连接浏览器」
3. 浏览并加载答案文件
4. 点击「扫描页面」，确认题目列表无误
5. 点击「开始答题」
6. 完成后点击「验证结果」，确认选中状态
7. 人工点击页面上的提交按钮

## 环境要求

- Python 3.10+
- Google Chrome
- 依赖见各子目录的 `requirements.txt`（主要为 `playwright` 和 `customtkinter`）

---

## 注意事项

- 所有工具均通过 CDP 连接浏览器，Chrome 须以调试模式运行（工具可自动完成此步骤）
- `swjtu-quiz-filler` 只负责选择，不会自动提交，请人工确认后再提交
- 答案文件行数须与页面题目顺序完全一致
